# AI Module

This folder is reserved for all AI-related code in ResumeIQ AI.

It will hold the modular intelligence layer for:

- prompt templates
- orchestration flows
- model routing policies
- semantic matching pipelines
- recruiter simulation logic
- truthfulness guardrails
- ATS optimization intelligence

As we build phase by phase, this folder will stay isolated from the UI in `apps/frontend/` and the API layer in `apps/backend/`, then be wired through clean service contracts.
